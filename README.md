# Project Report App

A React application built with Vite for submitting project reports.

## Features

- Form to submit project name, description, and status.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`.

## Build

To build the project for production:
```
npm run build
```

## Technologies Used

- React
- TypeScript
- Vite
